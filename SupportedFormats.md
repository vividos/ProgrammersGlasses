# Supported formats

## Developer formats

| Format | Extension        | Description | Support
| ------ | ---------------- | ----------- |
| COFF   | .obj             | COFF object file | ✅
|        | .obj             | non-COFF object file | partial
|        | .obj             | BigObj anonymous object file | partial
|        | .lib             | COFF library archive | ✅
| PE     | .exe, .dll, .sys | Portable Executable | partial
| MZ     | .exe             | MS-DOS Executable | partial

## Image formats

| Format | Extension        | Description | Support
| ------ | ---------------- | ----------- |
| PNG    | .png             | Portable Network Graphics file | partial

## Audio formats

| Format | Extension        | Description | Support
| ------ | ---------------- | ----------- |
| SID    | .sid, .psid      | C64 SID audio file | partial

## Miscellaneous formats

| Format | Extension        | Description | Support
| ------ | ---------------- | ----------- |
| D64    | .d64, .d71, .d81 | C64 disk image file | partial
